Aplikasi Penjualan Barang Berbasis WEB

Laravel 5

Aplikasi Penjualan Barang berbasis web ialah aplikasi yang dibangun menggunakan framework laravel. Aplikasi ini memiliki tampilan yang sangat simple dan fitur yang cukup lengkap untuk sebuah aplikasi Penjualan Barang, dan Aplikasi ini bisa langsung di aplikasikan, Aplikasi ini dapat dioperasikan secara online maupun offline dan cukup mudah untuk dijalankan.


Koleksi fitur yang ada dalam aplikasi Penjualan Barang ini ialah sebagai berikut :
- CRUD Barang, Distributor, Petugas
- Transaksi Penjualan
- Transaksi Barang Masuk
